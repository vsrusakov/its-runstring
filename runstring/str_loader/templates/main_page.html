Сайт для создания видео с бегущей строкой

Для создания бегущей строки введите /runstring после корневого каталога.
Можно передать необязательные параметры:

  • text - текст бегущей строки. По умолчанию "Hello, World!".
  • fps - число кадров в секунду. По умолчанию 24.
  • name - имя видеофайла. По умолчанию "my_video".

Пример запроса: http://localhost:8000/runstring?text=nice phrase!&fps=40
